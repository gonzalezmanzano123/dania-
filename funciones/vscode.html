¿Qué es VSC? -concepto

La marca Visual Studio siempre ha sido considerada como una de las más importantes a la hora de crear todo tipo de programas para Windows. Todo lo relacionado con Visual Studio, hace unos años, era sinónimo de código cerrado y, además, bastante caro. Por ello, la llegada de Visual Studio Code marcó un antes y un después para los programadores.

Poceso de instalacion

Como hemos dicho, Visual Studio Code es un programa totalmente gratuito y de código abierto. Por lo tanto, cualquier usuario interesado en usarlo, o solo en probarlo, puede bajarlo sin ninguna barrera. Para ello, lo que debemos hacer es acceder a este enlace, a su página principal donde encontraremos todas las versiones del programa.

  1. Paso. Instalar el software eMbedded Visual C++
  2. Paso. Actualice eMbedded Visual C++ a la última versión.
  3. Paso. Asociar EMbedded Visual C++ Project Format archivos con eMbedded Visual C++
  4. Paso. Asegúrese de que el archivo VCP esté completo y sin errores.
